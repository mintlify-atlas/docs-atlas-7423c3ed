## What is TradeMaster Transactions?

TradeMaster Transactions Portales is a powerful multi-tenant event ticketing system that enables organizations to create branded customer portals for event discovery, ticket purchasing, and payment processing. Built with React, Redux, and Firebase, it delivers a seamless experience for both event organizers and attendees.

## Built With Modern Technologies

TradeMaster Transactions leverages cutting-edge web technologies:

- **React 18** - Modern component-based UI framework
- **Redux Toolkit** - Predictable state management with encrypted persistence
- **Firebase** - Authentication, Firestore database, and Cloud Functions
- **Tailwind CSS** - Utility-first styling with Radix UI components
- **Stripe & Mercantil** - Secure payment processing
- **Vite** - Fast build tooling and hot module replacement
