## Overview

The Home view (`src/views/home/Home.js`) serves as the main landing page for the TradeMaster Transactions Portal. It displays an auto-rotating event carousel, featured upcoming events in a grid layout, and popular events sections.

**Route:** `/`

## Data Processing

### Event Sorting

Events are automatically sorted by date from nearest to farthest:

```javascript
const parseDate = (dateString) => {
  const [day, month, year] = dateString.split('/').map(Number);
  return new Date(year, month - 1, day);
};

const carouselItems = events.map((evento, i) => {
  return {
    id: i,
    image: evento.event_media[0] !== '' ? evento.event_media[0] : "https://placehold.co/1200x600/png",
    event_id: evento?.event_id,
    title: evento?.event_name,
    date: convertTimestamp(evento?.date_start),
  }
}).sort((a, b) => parseDate(a.date) - parseDate(b.date));
```

## Carousel Component

### Auto-Rotation Logic

```javascript
const Carousel = ({ items }) => {
  const [currentIndex, setCurrentIndex] = useState(0)

  useEffect(() => {
    const timer = setInterval(() => {
      nextSlide()
    }, 10000)  // Auto-advance every 10 seconds
    return () => clearInterval(timer)
  }, [])

  return (
    <div className="relative overflow-hidden rounded-lg h-[600px]">
      <div
        className="flex transition-transform duration-500 ease-in-out h-full"
        style={{ transform: `translateX(-${currentIndex * 100}%)` }}
      >
        {items.map((item) => (
          <div key={item.id} className="w-full flex-shrink-0 relative cursor-pointer" 
               onClick={() => { navigate(`event/${item.event_id}`) }}>
            <img src={item.image} alt={item.title} className="w-full h-full object-cover" />
            <div className="absolute inset-0 bg-gradient-to-r from-black/70 to-transparent" />
          </div>
        ))}
      </div>
      {/* Navigation buttons and indicators */}
    </div>
  )
};
```

### Features
- Auto-rotation every 10 seconds
- Manual navigation with prev/next buttons
- Click-to-navigate to event details
- Visual indicators for current slide
- Smooth CSS transitions

## Image Loading Pattern

### Progressive Image Loading

```javascript
const handleImageLoad = (id) => {
  setImageLoaded(prev => ({ ...prev, [id]: true }))
}

// In render:
<div className="absolute inset-0 flex items-center justify-center">
  {!imageLoaded[event.id] && <span className="text-gray-400">Cargando...</span>}
</div>
<img
  src={event.image}
  alt={event.title}
  className={`transition-opacity duration-300 ${imageLoaded[event.id] ? 'opacity-100' : 'opacity-0'}`}
  onLoad={() => handleImageLoad(event.id)}
/>
```

**Benefits:**
- Smooth loading transitions
- Loading indicators for better UX
- Prevents layout shifts
