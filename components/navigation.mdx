## DropdownMenu

A dropdown menu component built on Radix UI DropdownMenu primitive.

### Import

```javascript
import {
  DropdownMenu,
  DropdownMenuTrigger,
  DropdownMenuContent,
  DropdownMenuItem,
  DropdownMenuCheckboxItem,
  DropdownMenuRadioItem,
  DropdownMenuLabel,
  DropdownMenuSeparator,
  DropdownMenuShortcut,
  DropdownMenuGroup,
  DropdownMenuSub,
  DropdownMenuSubContent,
  DropdownMenuSubTrigger,
  DropdownMenuRadioGroup,
} from "@/components/ui/DropdownMenu"
```

### Component Architecture

DropdownMenu is composed of many parts for flexible composition:

- `DropdownMenu` - Root component
- `DropdownMenuTrigger` - Button that opens the menu
- `DropdownMenuContent` - The dropdown content (portaled)
- `DropdownMenuItem` - Individual menu item
- `DropdownMenuCheckboxItem` - Checkable menu item
- `DropdownMenuRadioItem` - Radio button menu item
- `DropdownMenuRadioGroup` - Groups radio items
- `DropdownMenuLabel` - Section label
- `DropdownMenuSeparator` - Visual separator
- `DropdownMenuShortcut` - Keyboard shortcut display
- `DropdownMenuGroup` - Groups related items
- `DropdownMenuSub` - Submenu container
- `DropdownMenuSubTrigger` - Opens submenu
- `DropdownMenuSubContent` - Submenu content

### Features

- **Keyboard navigation** - Arrow keys, Enter, Escape
- **Type-ahead** - Type to highlight items
- **Portal rendering** - Avoids z-index issues
- **Animations** - Smooth open/close transitions
- **Icons** - Check marks and chevrons from lucide-react
- **Submenus** - Nested menu support
- **Checkboxes & Radio** - State management built-in
- **Accessibility** - Full ARIA support

### Props

#### DropdownMenuItem

- `inset` - Adds left padding (for alignment with labeled items)
- `disabled` - Disables the item
- `onSelect` - Callback when item is selected

#### DropdownMenuCheckboxItem

- `checked` - Controlled checked state
- `onCheckedChange` - Callback when checked state changes

#### DropdownMenuRadioItem

- `value` - Value for radio item
- Must be used within `DropdownMenuRadioGroup`

## Tabs

A tabs component built on Radix UI Tabs primitive.

### Import

```javascript
import {
  Tabs,
  TabsList,
  TabsTrigger,
  TabsContent,
} from "@/components/ui/Tabs"
```

### Component Architecture

- `Tabs` - Root component
- `TabsList` - Container for tab triggers
- `TabsTrigger` - Individual tab button
- `TabsContent` - Content panel for each tab

### Component Source

```javascript
import * as React from "react"
import * as TabsPrimitive from "@radix-ui/react-tabs"
import { cn } from "../../utils/cn";

const Tabs = TabsPrimitive.Root

const TabsList = React.forwardRef(({ className, ...props }, ref) => (
  <TabsPrimitive.List
    ref={ref}
    className={cn(
      "inline-flex h-10 items-center justify-center rounded-md bg-muted p-1 text-muted-foreground",
      className
    )}
    {...props}
  />
))

const TabsTrigger = React.forwardRef(({ className, ...props }, ref) => (
  <TabsPrimitive.Trigger
    ref={ref}
    className={cn(
      "inline-flex items-center justify-center whitespace-nowrap rounded-sm px-3 py-1.5 text-sm font-medium ring-offset-background transition-all focus-visible:outline-none focus-visible:ring-2 focus-visible:ring-ring focus-visible:ring-offset-2 disabled:pointer-events-none disabled:opacity-50 data-[state=active]:bg-background data-[state=active]:text-foreground data-[state=active]:shadow-sm",
      className
    )}
    {...props}
  />
))

const TabsContent = React.forwardRef(({ className, ...props }, ref) => (
  <TabsPrimitive.Content
    ref={ref}
    className={cn(
      "mt-2 ring-offset-background focus-visible:outline-none focus-visible:ring-2 focus-visible:ring-ring focus-visible:ring-offset-2",
      className
    )}
    {...props}
  />
))
```

## ScrollArea

A custom scrollable area component built on Radix UI ScrollArea primitive.

### Import

```javascript
import { ScrollArea, ScrollBar } from "@/components/ui/ScrollArea"
```

### Component Source

```javascript
import * as React from "react";
import * as ScrollAreaPrimitive from "@radix-ui/react-scroll-area";
import { cn } from "../../utils/cn";

const ScrollArea = React.forwardRef(({ className, children, ...props }, ref) => (
  <ScrollAreaPrimitive.Root
    ref={ref}
    className={cn("relative overflow-hidden", className)}
    {...props}
  >
    <ScrollAreaPrimitive.Viewport className="h-full w-full rounded-[inherit]">
      {children}
    </ScrollAreaPrimitive.Viewport>
    <ScrollBar />
    <ScrollAreaPrimitive.Corner />
  </ScrollAreaPrimitive.Root>
));

const ScrollBar = React.forwardRef(
  ({ className, orientation = "vertical", ...props }, ref) => (
    <ScrollAreaPrimitive.ScrollAreaScrollbar
      ref={ref}
      orientation={orientation}
      className={cn(
        "flex touch-none select-none transition-colors",
        orientation === "vertical" && "h-full w-2.5 border-l border-l-transparent p-[1px]",
        orientation === "horizontal" && "h-2.5 flex-col border-t border-t-transparent p-[1px]",
        className
      )}
      {...props}
    >
      <ScrollAreaPrimitive.ScrollAreaThumb className="relative flex-1 rounded-full bg-border" />
    </ScrollAreaPrimitive.ScrollAreaScrollbar>
  )
);
```

### Features

- **Custom scrollbars** - Styled to match your design system
- **Vertical & Horizontal** - Both orientations supported
- **Native feel** - Smooth scrolling like native scrollbars
- **Cross-browser** - Consistent appearance across browsers
- **Accessibility** - Keyboard scrolling support

### Props

#### ScrollArea

- `className` - Additional CSS classes
- `children` - Scrollable content

#### ScrollBar

- `orientation` - `"vertical"` or `"horizontal"` (default: vertical)
- `className` - Additional CSS classes
